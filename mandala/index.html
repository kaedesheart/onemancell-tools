出力


  









  
  

    
使い方 — 曼荼羅タスク分解


    

      
操作

      

        タップ
        マスを編集（タスクを入力）
      

      

        中心へスワイプ
        そのタスクの中に潜る（さらに8つに分解）
      

      

        中心セルをタップ
        一つ上の階層に戻る
      

      

        Esc
        上に戻る / 出力パネルを閉じる
      

    


    

      
編集

      

        Enter
        確定して保存
      

      

        空白で確定
        そのマスを削除
      

      

        時間ボタン
        見積もり時間を設定（less / 5m / 30m / more）
親のマスには合計が自動表示されます
      

    


    

      
表示

      

        アウトライン
        全タスクをツリー形式で確認・編集
      

      

        出力
        プレーン / Markdown でクリップボードにコピー
      

      

        新規
        新しいプロジェクトを開始（データは消えます）
      

    


    
データはブラウザに自動保存されます。
