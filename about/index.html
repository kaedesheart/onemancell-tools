About

onemancell tools は、日常をちょっと便利・楽しくする小さな Web ツールを作って公開している、個人運営のサイトです。

サイトについて

「onemancell」とは、運営者の活動名です。「ひとつの細胞のように、小さくても自分の手で考えて作る」をコンセプトに、個人で企画・開発したツールを公開しています。

当サイトのツールはすべて、ブラウザ上で完結するクライアントサイドアプリケーションです。サーバーへのデータ送信はなく、利用にあたって会員登録などは不要です。

提供しているツール

2026年4月時点で 5 つのツールを公開しています。

🎲
エモクロアダイスツール
エモクロア TRPG のための判定・ステータス管理。各種ダイスロールに対応。
🧩
曼荼羅タスク分解
3×3 のグリッドで思考を整理。大きな目標を小さなタスクに細分化する手助けに。
🍅
ポモドーロタイマー
25分集中・5分休憩のシンプルなタイマー。スリープ防止つきで集中をサポート。
🌙
のんびりノイズ
雨・波・風・焚き火を Web Audio API で合成するノイズミキサー。プリセット保存可。
🪐
スイングバイ
惑星の重力で弾道を曲げる宇宙パズル。全6ステージを最少発射数でクリアを目指す。

開発について

当サイトのツールはすべて軽量なバニラ JavaScript で実装されています。フレームワークやビルドツールに依存しないため、ロードが速く、また将来にわたってメンテナンスしやすい構成を心がけています。

各ツールは PWA（プログレッシブ Web アプリ）に対応しており、スマートフォンの「ホーム画面に追加」機能を使うことで、ネイティブアプリのように利用できます。
